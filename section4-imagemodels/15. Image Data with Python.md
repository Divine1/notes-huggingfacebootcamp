opencv assumes BGR color code. matplotlib assumes RGB.
```
img_rgb = cv2.cvtColor(img,cv2.COLOR_BGR2RGB)
plt.imshow(img_rgb)

img_gray = cv2.imread('puppy.png',cv2.IMREAD_GRAYSCALE) # convert to grayscale
plt.imshow(img_gray)

plt.imshow(img_gray,cmap='gray')

#resize with pixcel
```
